# ☕ Brew Recipes — وصفات القهوة المختصة

متجر رقمي فاخر لوصفات القهوة المختصة، مبني بـ Next.js 16 و React 19 و Prisma و Tailwind CSS 4.

## المزايا

- 🏠 **الرئيسية**: قسم بطولي فاخر، الوصفات المميزة، التصنيفات حسب طريقة التحضير، آراء العملاء، الأسئلة الشائعة
- 🛒 **المتجر**: بحث وفلترة حسب نوع التحضير ودرجة التحميص، وترتيب حسب السعر
- 📖 **صفحة الوصفة**: مواصفات كاملة، خطوات التحضير، ملاحظات التذوق، وحاسبة ذكية تعيد حساب الماء والوقت عند تغيير كمية البن
- ❤️ **المفضلة**: أضف أي وصفة لقائمتك بضغطة قلب
- 👑 **الاشتراكات**: خطة شهرية وسنوية مع تفعيل فوري (وضع تجريبي)
- 👤 **الحساب**: تسجيل دخول/حساب جديد، ملف شخصي يعرض المشتريات والمفضلة والاشتراك
- 🛠️ **لوحة التحكم**: إحصائيات حية، إضافة/تعديل/حذف الوصفات بنموذج كامل
- 🌗 وضع فاتح/داكن + تبديل اللغة عربي/إنجليزي (RTL/LTR)

## التشغيل

```bash
npm install        # تثبيت الاعتماديات
npm run setup      # توليد Prisma + إنشاء قاعدة البيانات + بذر الوصفات
npm run dev        # تشغيل خادم التطوير على http://localhost:3000
```

قاعدة البيانات SQLite محلية (`prisma/dev.db`) — لا تحتاج أي إعدادات خارجية.

### الدخول كمدير (وضع تجريبي)

سجّل الدخول بالبريد `admin@brew.com` وأي كلمة مرور (٦ أحرف فأكثر) لتظهر لك لوحة التحكم.

## الأوامر

| الأمر | الوظيفة |
|---|---|
| `npm run dev` | خادم التطوير |
| `npm run build` | بناء نسخة الإنتاج |
| `npm run start` | تشغيل نسخة الإنتاج |
| `npm run lint` | فحص الكود |
| `npm run db:generate` | توليد عميل Prisma |
| `npm run db:push` | مزامنة المخطط مع قاعدة البيانات |
| `npm run db:seed` | بذر الوصفات التجريبية |
| `npm run setup` | الثلاثة السابقة معاً |

## البنية

```
app/
  page.tsx                    الرئيسية (Server Component — بيانات حقيقية)
  store/                      المتجر
  recipe/[slug]/              صفحة الوصفة
  favorites/                  المفضلة
  subscriptions/              الاشتراكات
  login/ register/ profile/   الحساب
  admin/                      لوحة التحكم (قائمة/إضافة/تعديل/حذف)
  api/recipes/                REST API للوصفات
  components/                 المكونات المشتركة
lib/
  prisma.ts                   عميل Prisma
  recipes.ts                  طبقة البيانات (تحويل JSON ↔ مصفوفات)
  app-store.ts                حالة المتصفح (zustand): الحساب، المفضلة، المشتريات، الاشتراك
  admin-actions.ts            Server Actions للوحة التحكم
prisma/
  schema.prisma               المخطط (SQLite)
  seed.ts                     البذر
```

## ملاحظات الوضع التجريبي

- **المصادقة** محلية على المتصفح (zustand + localStorage). ملفات `lib/supabase/` جاهزة للربط بـ Supabase Auth عند الإطلاق.
- **الدفع** يُفعَّل فوراً بدون بوابة دفع. نموذج `Payment` في المخطط جاهز لربط Stripe.
- **لوحة التحكم** يظهر رابطها للمدير فقط، لكن حمايتها الفعلية تتطلب مصادقة من جهة الخادم — تُضاف مع Supabase.
